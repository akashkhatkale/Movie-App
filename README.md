## Movie App is a TMDB client that uses modern android development tools and techniques such as 
1. Jetpack compose
2. Unit tests 
3. Github Actions
4. Dagger
5. Lifecycle aware components etc
6. Coroutines and flows
7. MVVM
8. Retrofit
9. RoomDB
